# @vaadin/stepper

A web component for displaying a step-by-step workflow.

> ⚠️ This component is experimental and the API may change. In order to use it, enable the feature flag by setting `window.Vaadin.featureFlags.stepperComponent = true`.

[Documentation + Live Demo ↗](https://vaadin.com/docs/latest/components/stepper)

[![npm version](https://badgen.net/npm/v/@vaadin/stepper)](https://www.npmjs.com/package/@vaadin/stepper)

```html
<vaadin-stepper>Example</vaadin-stepper>
```

[<img src="https://raw.githubusercontent.com/vaadin/web-components/main/packages/stepper/screenshot.png" width="296" alt="Screenshot of vaadin-stepper">](https://vaadin.com/docs/latest/components/stepper)

## Installation

Install the component:

```sh
npm i @vaadin/stepper
```

Once installed, import the component in your application:

```js
import '@vaadin/stepper';
```
