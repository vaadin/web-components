## Installation

Install the iconset:

```sh
npm i @vaadin/icons
```

Once installed, import it in your application:

```js
import '@vaadin/icons';
```

To display an icon, use [`<vaadin-icon>`](https://www.npmjs.com/package/@vaadin/icon) element:

```html
<vaadin-icon icon="vaadin:phone"></vaadin-icon>
```
