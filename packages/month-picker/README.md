# @vaadin/month-picker

> ⚠️ **This component is experimental.** Enable the feature flag before importing:
>
> ```js
> window.Vaadin ||= {};
> window.Vaadin.featureFlags ||= {};
> window.Vaadin.featureFlags.monthPickerComponent = true;
> ```

An input field that allows the user to select a month and a year.

[Documentation + Live Demo ↗](https://vaadin.com/docs/latest/components/month-picker)

[![npm version](https://badgen.net/npm/v/@vaadin/month-picker)](https://www.npmjs.com/package/@vaadin/month-picker)

```html
<vaadin-month-picker label="Month"></vaadin-month-picker>
```

## Installation

Install the component:

```sh
npm i @vaadin/month-picker
```

Once installed, import the component in your application:

```js
import '@vaadin/month-picker';
```
