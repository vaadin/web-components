# Research Sources

External design system libraries to research for component API design patterns.

## Libraries

- Material UI (MUI) — https://mui.com
- Ant Design — https://ant.design
- Chakra UI — https://chakra-ui.com
- Web Awesome — https://webawesome.com
- Base UI — https://base-ui.com
- Carbon Design System (IBM) — https://www.carbondesignsystem.com
- Radix UI — https://radix-ui.com
- React Aria (Adobe) — https://react-aria.adobe.com

## What to Look For

- Developer-facing API shape: how developers configure, pass data, handle events, customize appearance
- Patterns that minimize boilerplate for common cases
- Progressive disclosure: simple defaults, opt-in complexity
